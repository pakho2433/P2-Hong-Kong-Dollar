# P2 Hong Kong Dollar

適合小學二年級學生在 iPad、手機及電腦使用的 3D 香港貨幣購物學習遊戲。

## 玩法

- 使用左下角搖桿在大型 3D 公園自由行走。
- 直接拖動 3D 畫面旋轉視角，行走期間亦可同時轉動鏡頭。
- 向媽媽接購物任務，再到六間店舖選購指定貨品。
- 媽媽會提供多於所需的金額，學生要自行選擇正確的香港紙幣及硬幣付款。
- 買齊物品後返回媽媽交付任務。

## 六間店舖

零食店、公仔店、氣球店、飲品店、水果店、文具店。

## P2 更新內容

- 公園範圍及公園地圖加大。
- 增加樹林、灌木及遠景，減少看到場景邊界。
- 加入蝴蝶、雲朵、氣球和噴水池動畫。
- 支援行走期間同時拖動畫面調整視角。
- 修正上下視角控制方向。

遊戲需要網絡連線以載入 Three.js 及香港金融管理局貨幣參考圖片。
